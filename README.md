## Install

```bash
npm install
git clone https://github.com/inteldev2080/react-native-instagram.git
cd InstagramClone

```
## Usage

### Android
```bash
npx react-native start
npx react-native run-android
```
### IOS
```bash
npx react-native start
cd ios && pod install
npx react-native run-ios
```
